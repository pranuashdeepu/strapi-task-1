Project Overview

This repository contains a **Strapi CMS application** created as part of a hands-on setup task.  
The project demonstrates installing Strapi, configuring the admin panel, creating a custom content type, and publishing content using the Strapi Admin UI.

---

 Technologies Used

- **Strapi v5.34.0**
- **Node.js v24.13.0**
- **npm v11.x**
- **SQLite (default database)**
- **JavaScript / TypeScript**
- **Git & GitHub**
- **VS Code**
